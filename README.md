## 💻 Shell Scripts Overview

### 🔹 Script 1 — System Identity Report
Displays system information such as OS, kernel version, user, uptime, date, and license.

---

### 🔹 Script 2 — FOSS Package Inspector
Checks if a package (Python) is installed, displays its version, and prints a short description using conditional statements.

---

### 🔹 Script 3 — Directory and Permission Auditor
Iterates through files in a directory and checks their read, write, and execute permissions.

---

### 🔹 Script 4 — File Analyzer
Analyzes a given file and displays its size, number of lines, and number of words using functions and arguments.

---

### 🔹 Script 5 — Temporary File Cleaner
Removes temporary files older than a specified number of days from the system.

